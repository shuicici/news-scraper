# News Scraper – Headlines & Articles

Scrape news articles and headlines from major news sources. Get structured data for media monitoring, sentiment analysis, and market research.

## What it extracts

- Article titles and headlines
- Publication dates and sources
- Article summaries and excerpts
- Author information
- Category and topic tags

## Use cases

- **Media monitoring** — Track news about specific topics or brands
- **Sentiment analysis** — Analyze tone and coverage of events
- **PR research** — Understand media landscape for campaigns
- **Market research** — Track industry news and trends

## Notes

- Scrapes publicly available news articles
- Supports multiple news sources
- No API key required

## Input

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| query | string | Yes | Search keyword or topic |
| limit | number | No | Max articles to extract (default: 20) |
